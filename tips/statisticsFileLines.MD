## 统计文件行数

经常需要统计文件有多少行，或者某个文件夹下的文件分别以及一共有多少行，例如统计代码行数的时候，当在linux上的时候去查看就比较麻烦，下面这个命令可以统计文件行数

例如我要统计当前目录下所有markdown文件的行数

```
find . "(" -name "*.md" -or -name "*.MD" ")" -print | xargs wc -l
```

它会输出:

```
      33 ./密码.md
      75 ./paste.md
      60 ./sendEmail.MD
     168 total
```

这些就分别是对应文件的行数和总行数。
